🛒 Quick Shop – E-Commerce Web App

Welcome to Quick Shop – a simple, clean, and responsive e-commerce web application built to simulate the online shopping experience. Users can browse products, add items to the cart, and complete a checkout process.

🔧 Features

🔐 Authentication

-> Login form with email and password

-> Signup route for new user registration

-> Basic session/token simulation (non-persistent demo)

🛍️ Product Listing

-> Static list of products rendered on the homepage

-> "Add to Cart" buttons for each item

🧺 Shopping Cart

-> Add/remove products dynamically

-> Displays total price

📦 Checkout

-> User enters name, address, email

-> Confirmation popup on successful order placement
